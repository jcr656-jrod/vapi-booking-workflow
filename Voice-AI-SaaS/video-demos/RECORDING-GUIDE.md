# Video Demo Recording Guide

## Overview

Record one demo video per industry. Total time: ~3-4 hours for all 29 industries.

---

## Equipment Needed

- **Screen recorder**: Loom (free), OBS, or QuickTime
- **Microphone**: Any decent mic (laptop mic works)
- **Phone**: To make demo calls
- **Quiet room**: Minimize background noise

---

## Recording Setup

### Option A: Loom (Easiest)
1. Install Loom Chrome extension
2. Click Loom icon → Screen + Cam
3. Select screen to record
4. Start recording

### Option B: OBS (Free, More Control)
1. Download OBS Studio
2. Add "Display Capture" source
3. Add "Audio Input Capture" for mic
4. Click "Start Recording"

---

## Demo Script Template

Each video should follow this structure (60-90 seconds):

```
[0:00-0:10] INTRO
"Hi, this is Sarah from [Business Name]. How can I help you today?"

[0:10-0:25] CUSTOMER REQUEST
Caller: "I'd like to schedule an appointment"
AI: "I'd be happy to help! What day works best for you?"

[0:25-0:40] AVAILABILITY CHECK
Caller: "How about Thursday?"
AI: "Let me check Thursday's availability... I have openings at 10 AM, 2 PM, and 4 PM."

[0:40-0:55] BOOKING
Caller: "2 PM works"
AI: "Perfect! Can I get your name and phone number?"
[Collect info]

[0:55-1:10] CONFIRMATION
AI: "You're all set! I've booked you for Thursday at 2 PM. You'll receive a confirmation text shortly."

[1:10-1:20] OUTRO
AI: "Is there anything else I can help you with today?"
Caller: "No, that's all. Thank you!"
AI: "Thank you for calling [Business Name]. Have a great day!"
```

## Industry-Specific Scripts

### ACCOUNTING - Precision Accounting Services
```
Intro: "Thank you for calling Precision Accounting Services. This is your AI assistant. How can I help you today?"
Services to mention: Tax preparation, bookkeeping consultation, quarterly review
Appointment types: "Tax consultation" or "Bookkeeping review"
```

### AUTOMOTIVE - First Coast Auto Repair
```
Intro: "Thanks for calling First Coast Auto Repair. How can I help you?"
Services: Oil change, brake inspection, engine diagnostics
Appointment types: "Service appointment" or "Diagnostic check"
Add: "What type of vehicle do you have?"
```

```
Intro: "Thank you for calling Little Stars Learning Center!"
Services: Daycare, preschool, after-school programs
Appointment types: "Tour" or "Enrollment consultation"
Add: "How old is your child?"
```

### CLEANING - Sparkle Clean Services
```
Intro: "Thanks for calling Sparkle Clean Services!"
Services: Home cleaning, deep cleaning, move-in/out cleaning
Appointment types: "Free estimate" or "Cleaning appointment"
```

### DENTAL - Sunshine Dental Care
```
Intro: "Thank you for calling Sunshine Dental Care!"

```

## Recording Checklist

For each industry video:

- [ ] Open demo page in browser (visible in recording)
- [ ] Start screen recording
- [ ] Call VAPI demo number
- [ ] Follow script naturally (don't sound robotic)
- [ ] Show confirmation SMS/email arriving (if possible)
- [ ] Stop recording
- [ ] Trim intro/outro dead air
- [ ] Upload to YouTube/Loom
- [ ] Copy embed code

## Upload Locations

### YouTube (Recommended)
1. Upload as "Unlisted" (not public, but embeddable)
2. Add title: "[Industry] - AI Receptionist Demo | Voice AI SaaS"
3. Add description with CTA
4. Copy embed code

### Loom (Easier)
1. Finish recording → Auto-uploads
2. Click "Share" → "Embed"
3. Copy embed code
