# Voice-AI SaaS

**White-Label Voice AI Booking System**

Turn any business into an AI-powered appointment booking machine. This template allows you to deploy voice AI booking systems for multiple clients with custom branding.

---

## Features

- **Voice AI Booking** - Customers call, AI handles scheduling
- **Google Calendar Integration** - Real-time availability & booking
- **CRM Integration** - Auto-create contacts & opportunities in GoHighLevel
- **SMS Confirmations** - Instant text confirmations to customers
- **Email Notifications** - Branded emails to both customer and business owner
- **Conflict Detection** - Prevents double-booking
- **Google Meet Links** - Auto-generated for virtual appointments

---

## Quick Start

### 1. Clone the Template Workflow
Import `workflow-template.json` into your n8n instance.

### 2. Configure Client Settings
Update the `CLIENT_CONFIG` code node with client details.

### 3. Set Up Credentials
- Google Calendar OAuth
- SMTP Email
- GoHighLevel API (optional)

### 4. Connect to VAPI
Add webhook URLs to VAPI assistant tools.

### 5. Activate & Test
